# Trailhead Leaderboard

A competitive leaderboard platform for tracking Salesforce Trailhead progress.

## Features
- Real-time scraping of Trailhead profiles
- Agentblazer status tracking (Champion, Innovator, Legend 2026)
- Certification tracking
- Excel import/export
- Cyberpunk UI design

## Tech Stack
- **Frontend**: React + Vite
- **Backend**: FastAPI + Python
- **Database**: MongoDB
- **Scraping**: Playwright
